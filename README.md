# Hello World TypeScript Project

This is a simple Hello World project written in TypeScript.

## How to Run

1. Install dependencies:
   ```sh
   npm install
   ```
2. Build the project:
   ```sh
   npm run build
   ```
3. Run the project:
   ```sh
   npm start
   ```

## Project Structure
- `src/index.ts`: Main entry point
- `dist/`: Compiled JavaScript output
